# IRINATWE Bright Portfolio

A personal creative portfolio website for IRINATWE Bright and THEE GALLERY Creative Studio.

## Overview

This site showcases featured work, core services, studio profile, and contact details.

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript (ES6+)
- Google Fonts (Bebas Neue, Instrument Serif, DM Sans)

## Features

- Responsive design for mobile and desktop
- Smooth scroll navigation
- Interactive portfolio gallery
- Contact form integration
- Social media links
- Multi-page work showcase

## Project Structure

- `index.html` - Main homepage
- `irinatwe-bright-portfolio.css` - Site styles and responsive layout
- `irinatwe-bright-portfolio.js` - Interactive behavior (navigation, reveal effects)
- `irinatwe-bright-portfolio.html` - Alternative portfolio page (legacy)
- `all-work/` - Extended work pages
  - `all-work.html` - Overview of all work
  - `brand.html` - Brand identity projects
  - `events.html` - Event design projects
  - `sports.html` - Sports graphics projects
  - `work.css` - Styles for work pages
  - `work.js` - JavaScript for work pages
- `Images/` - Portfolio and profile images
  - `All the Work/` - General work images
  - `Brand Identity/` - Brand-related images
  - `Events/` - Event design images
  - `Music/` - Music-related images
  - `Profile/` - Profile and personal images
  - `Sports/` - Sports graphics images

## Run Locally

1. Open the project folder in VS Code.
2. Open `index.html` in your browser.
